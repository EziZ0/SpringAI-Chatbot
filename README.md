# 🤖 Gemini Chatbot using Spring AI

A simple AI-powered chatbot built with **Spring Boot**, **Spring WebFlux**, **Spring AI**, **React**, and **Google Gemini**. This project demonstrates how to integrate Google's Gemini Large Language Model (LLM) into a reactive Spring Boot application with a modern React frontend.


## 🛠️ Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring WebFlux
- Spring AI
- Google Gemini API
- Maven

### Frontend
- React
- JavaScript
- Bootstrap


## 💡 How It Works

1. The user enters a message in the React application.
2. The request is sent to the Spring WebFlux backend.
3. Spring AI forwards the prompt to the Google Gemini model.
4. Gemini generates an AI response.
5. The backend returns the response to the React frontend, where it is displayed in the chat interface.
